all of the tools used for RX 4XX-5XX vBIOS modding

provided by avkila on discord
@avkilasmodding on instagram

description of tools :

amdvbflash : lets you flash vBIOS or read vBIOS

SRBPolaris : easy to use bios modding tool. supports many things.

AMD GOP 1.6X.1.50_signed files : modified AMDGOP driver on your vBIOS so you can still UEFI boot on modded vBIOS
you MUST disable secure boot. or else your system wont post/give black display.

atimdag-patcher : patches your drivers so it doesnt do code43 error on desktop after driver install. must have.

PolarisBiosEditor : supports 460, 470D, 470, 480, 550, 560, 560XT, 570, 580 2048sp, 580, 590.
probably the easiest bios editing tool to ever exist.

Red BIOS Editor : bios modding tool for changing min max voltage, vram voltage, and many other stuff. better than pbe in my opinion.

HxD : hex editor so you can inject your modded GOP driver into the .rom file. Be careful while using, you may brick your gpu.
